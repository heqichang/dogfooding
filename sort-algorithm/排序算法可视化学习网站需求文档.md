# 排序算法可视化学习网站需求文档

基于 React \+ TypeScript \+ Vite 开发，面向程序员的交互式排序算法学习平台，支持动态演示、步骤控制、源码对照、复杂度分析、对比测试，覆盖主流 \+ 进阶全排序算法。

## 一、核心定位

面向程序员 / 学生的**交互式排序算法可视化学习平台**，支持**动态演示、步骤控制、源码对照、复杂度分析、对比测试**，覆盖主流 \+ 进阶全排序算法。

## 二、完整功能清单

### 1\. 排序算法可视化（增强版）

- 支持柱状图 / 折线图 / 色块三种可视化样式

- 单步执行 / 逐步自动播放 / 全速运行三种模式

- 可调节动画速度（慢 / 中 / 快 / 极速）

- 支持暂停 / 继续 / 重置排序过程

- 高亮显示：

    - 当前比较的元素

    - 当前交换的元素

    - 已排序完成区域

    - 未排序区域

- 支持自定义数组：手动输入、随机生成、逆序、近乎有序

- 支持数组长度动态调整（10\~200）

### 2\. 算法理论知识（增强版）

- 算法简介（一句话说明）

- 优点 / 缺点 / 适用场景

- 标准改进方案（如快排优化、归并非递归、二分插入排序）

- 时间复杂度（最好 / 最坏 / 平均）

- 空间复杂度

- 稳定性说明（稳定 / 不稳定）

- 是否原地排序

- 真实业务使用场景举例

### 3\. 算法覆盖范围

#### 基础排序

- 冒泡排序 Bubble Sort

- 选择排序 Selection Sort

- 插入排序 Insertion Sort

#### 进阶排序（面试高频）

- 希尔排序 Shell Sort

- 归并排序 Merge Sort

- 快速排序 Quick Sort

- 堆排序 Heap Sort

#### 高级 / 特殊排序（拓展）

- 计数排序 Counting Sort

- 桶排序 Bucket Sort

- 基数排序 Radix Sort

- 鸡尾酒排序

- 梳排序

- 二分插入排序

- 三路快速排序

### 4\. 源码对照学习（新增）

- 每种算法提供 TypeScript 源码

- 代码附带详细注释

- 可视化步骤与代码执行位置联动高亮

- 支持一键复制源码

- 可选支持 JS / Java / Python 多语言版本

### 5\. 算法对比功能（新增）

- 同时选择 2\~4 种排序并行可视化

- 实时对比执行时间、比较次数、交换次数

- 自动生成对比结论

- 支持相同数据集、不同数据集对比

### 6\. 交互与学习模式（新增）

- 学习模式：一步步讲解算法流程

- 考试模式：隐藏下一步，让用户预测比较 / 交换位置

- 自由调试模式：可手动修改数组、断点执行

### 7\. 数据统计面板（新增）

- 比较次数

- 交换 / 移动次数

- 运行时间

- 时间复杂度理论值 vs 实际运行趋势

### 8\. 键盘快捷键（新增）

- 空格：播放 / 暂停

- 方向右键：单步前进

- R：重置

- 数字 1\~9：快速切换算法

- S：减速 / F：加速

### 9\. UI / 体验增强（新增）

- 深色 / 浅色模式

- 响应式布局（手机 / 平板 / PC）

- 流畅动画（Vite \+ React 性能优化）

- 美观图表、布局、字体

- 支持全屏演示

### 10\. 学习路径与引导（新增）

- 新手入门引导

- 算法难度分级（初级 / 中级 / 高级）

- 推荐学习顺序

- 各算法常见面试题总结

### 11\. 导出与分享（新增）

- 导出排序过程截图

- 生成分享链接（可复现当前数组与排序状态）

- 导出算法笔记（文本格式）

### 12\. 性能测试模式（新增）

- 大数据量排序性能压测（1000/1 万 / 10 万数据）

- 展示不同算法在不同数据规模下的表现

- 图表展示性能对比曲线

## 三、技术栈

- 框架：React 18

- 语言：TypeScript

- 构建工具：Vite

- 样式方案：Tailwind CSS / CSS Modules

- 动画：CSS Transition \+ React 状态控制

- 图标：React Icons

- 代码高亮：react\-syntax\-highlighter

- 状态管理：React useState \+ useReducer

## 四、页面结构

1. 首页：算法列表 \+ 学习入口

2. 算法详情页：可视化 \+ 说明 \+ 源码

3. 对比页面：多算法同时运行对比

4. 学习指南：排序算法学习路线 \+ 面试题

5. 设置页：主题、速度、可视化样式配置

## 五、项目介绍

基于 React \+ TypeScript \+ Vite 开发的排序算法可视化学习平台，提供动态演示、单步调试、源码对照、复杂度分析、多算法对比，覆盖全种类排序算法，帮助程序员直观理解原理、提升数据结构与算法能力。
