# 打字练习Web程序需求文档

## 一、文档概述

### 1\.1 文档目的

本文档明确**Web 版打字练习工具** 的功能需求、非功能需求、界面设计、交互逻辑等核心内容，为开发、测试、验收提供统一标准，最终实现一款简洁、流畅、实用的在线打字练习工具，满足用户基础打字练习、速度测试、成绩统计等核心需求。

### 1\.2 项目背景

核心价值为**零基础打字入门、指法练习、速度提升**，无复杂功能、上手即用。本项目基于 Web 技术复刻其核心体验，适配电脑端浏览器，无需安装、开箱即用，覆盖新手入门、速度测试、自定义练习等场景。

### 1\.3 目标用户

1. 打字零基础用户（学习键盘布局、标准指法）

2. 学生 / 办公人群（提升打字速度、准确率）

3. 轻量化打字练习需求用户（无广告、无复杂操作）

## 二、整体产品定位

1. 核心定位：**简洁免费、开箱即用的 Web 端打字练习工具**

2. 核心优势：无安装、无广告、功能聚焦、适配全浏览器、数据本地留存

3. 核心功能：指法练习、英文 / 中文打字、速度测试、实时统计、历史成绩

4. 界面风格：极简清爽、高对比度、专注打字本身（类 TT 打字复古简洁风）

## 三、功能需求

### 3\.1 核心功能模块

整体分为 5 大核心模块，覆盖打字练习全流程：

1. 练习模式模块

2. 实时打字检测模块

3. 成绩统计与展示模块

4. 键盘指法提示模块

5. 本地数据管理模块

---

### 3\.2 详细功能说明

#### 3\.2\.1 练习模式（核心功能）

支持 3 种经典打字练习模式，满足不同练习需求：

1. **键位指法练习（新手必备）**

    - 练习内容：26 个英文字母、数字 0\-9、常用符号（逗号、句号、空格等）

    - 规则：单键 / 双键循序练习，强制匹配正确按键才能继续

    - 辅助：实时显示标准指法图示，提示当前按键对应手指

2. **英文文章练习**

    - 练习内容：预置小学 / 初中 / 高中分级英文短文、常用单词句

    - 规则：逐词匹配，错误字符标红，支持回删修正

3. **中文文章练习**

    - 练习内容：预置经典课文、日常短句、办公常用文案

    - 适配：支持简体中文，匹配主流输入法输入逻辑

#### 3\.2\.2 实时打字检测（核心交互）

1. 输入匹配：用户输入字符与目标文本**实时对比**，正确字符标灰 / 绿色，错误字符标红

2. 光标跟随：输入光标实时跟随当前打字位置，不可跳跃打字

3. 错误处理：错误字符不跳过，必须删除修正后才能继续（可配置宽松模式）

4. 计时逻辑：

    - 按下第一个按键时开始计时

    - 完成所有文本输入时自动停止计时

5. 进度展示：实时显示打字进度（如 35/100 字符）

#### 3\.2\.3 成绩统计与展示

练习结束后自动生成成绩报告，核心指标：

1. 速度指标：**CPM（字符 / 分钟）**、**WPM（单词 / 分钟）**

2. 准确率：正确字符数 / 总输入字符数 ×100%

3. 错误统计：总错误数、错误字符列表

4. 用时：总练习时长（精确到秒）

5. 等级评价：根据速度 \+ 准确率评级（如：入门、熟练、高手）

#### 3\.2\.4 键盘指法提示

1. 静态指法图：首页展示标准键盘指法分区（左手 / 右手、食指 / 中指 / 无名指 / 小指对应键位）

2. 实时提示：练习时高亮当前需要按下的按键，标注对应手指

3. 新手模式：默认开启指法提示，可手动关闭

#### 3\.2\.5 本地数据管理

1. 无需注册登录，数据存储在浏览器本地（LocalStorage）

2. 历史成绩：记录近 30 次练习记录（模式、时间、速度、准确率、用时）

3. 数据导出：支持将历史成绩导出为文本文件

4. 数据清空：支持一键清空本地练习记录

#### 3\.2\.6 基础设置功能

1. 文本切换：支持手动切换练习文章 / 键位内容

2. 模式切换：宽松模式（错误可跳过）/ 严格模式（必须修正错误）

3. 界面设置：字体大小切换、深色 / 浅色模式切换

4. 提示开关：指法提示、打字音效开关

---

### 3\.3 非核心辅助功能

1. 打字音效：正确 / 错误输入对应不同提示音（可关闭）

2. 全屏模式：一键进入全屏，屏蔽干扰，专注练习

3. 响应式适配：支持电脑端所有浏览器（Chrome/Firefox/Edge/ Safari）

4. 无广告、无弹窗、无需注册

## 四、非功能需求

### 4\.1 性能需求

1. 加载速度：页面首次加载≤2 秒，功能切换无延迟

2. 运行流畅：输入实时响应，无卡顿、无延迟

3. 兼容性：支持所有现代浏览器，无需依赖插件

### 4\.2 易用性需求

1. 上手门槛：打开网页即可练习，无需教程

2. 操作极简：所有功能按钮≤5 个，界面无冗余元素

3. 视觉清晰：文字对比度高，长时间练习不疲劳

### 4\.3 稳定性需求

1. 练习过程无崩溃、无数据丢失

2. 本地记录永久保存（清除浏览器缓存除外）

3. 错误输入无卡死、无逻辑异常

### 4\.4 安全性需求

1. 无用户隐私数据收集，纯本地运行

2. 无后端请求，无外链风险

3. 代码开源透明，无恶意脚本

## 五、界面设计需求

### 5\.1 整体风格

- 极简清爽、高可读性、复古打字风

- 主色调：白色 / 浅灰（底色）\+ 黑色（正文）\+ 绿色（正确）\+ 红色（错误）

- 无复杂图片、无动画干扰

### 5\.2 页面结构（单页面应用，仅 1 个主界面）

1. **顶部导航栏**

    - logo（极简文字：打字练习）

    - 功能按钮：模式切换、设置、历史成绩、全屏

2. **中间核心区域（90% 视觉重心）**

    - 上半部分：**目标练习文本**（最大字号，清晰展示）

    - 下半部分：**实时输入区域**（同步显示输入内容，标色区分对错）

    - 中央：实时光标、进度条

3. **底部统计栏（练习时显示）**

    - 实时速度、准确率、用时、进度

4. **弹窗模块**

    - 成绩报告弹窗（练习结束）

    - 历史成绩弹窗

    - 设置弹窗

    - 指法图示弹窗

### 5\.3 界面规范

1. 字体：等宽字体（如 Consolas、微软雅黑等宽），保证字符对齐

2. 字号：练习文本≥16px，核心数据≥18px

3. 间距：文本行间距≥1\.5，无拥挤感

4. 配色：

    - 正确文字：深灰色 / 绿色

    - 错误文字：鲜红色

    - 未输入文字：浅灰色

    - 背景：纯白 / 浅灰（护眼）

## 六、交互逻辑需求

### 6\.1 核心交互流程

1. 打开网页 → 默认显示「键位练习」\+ 指法提示

2. 用户选择练习模式 / 文本 → 进入练习界面

3. 按下第一个键 → 计时开始

4. 实时输入 → 系统实时匹配对错、高亮显示

5. 输入完成 → 自动停止计时 → 弹出成绩报告

6. 可重新练习、切换文本、查看历史记录

### 6\.2 细节交互规则

1. 禁止复制粘贴文本（保证练习真实性）

2. 仅支持键盘输入，鼠标仅用于切换功能

3. 严格模式下：错误字符必须按删除键修正，才能继续输入

4. 回车、空格为有效字符，参与统计

5. 退出全屏 / 刷新页面 → 终止当前练习，不保存成绩

## 七、开发技术栈（前端纯静态，无后端）

1. 结构：HTML5

2. 样式：CSS3 / 纯 CSS（无复杂框架）

3. 逻辑：原生 JavaScript（无框架，轻量化）

4. 存储：浏览器 LocalStorage

5. 适配：纯前端响应式布局

6. 部署：静态网页（可直接打开 HTML 文件运行）
